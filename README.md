📦 Sistema de Inventario en Python

🧠 Descripción del proyecto:

Este proyecto consiste en el desarrollo de un sistema de inventario en consola utilizando Python.  
Permite gestionar productos mediante un menú interactivo, aplicando conceptos fundamentales de programación.

---

🚀 Funcionalidades

El programa permite:

- ➕ Agregar productos al inventario  
- 📋 Mostrar todos los productos registrados  
- 📊 Calcular estadísticas del inventario:
  - Valor total (precio × cantidad)
  - Cantidad total de productos  
- ❌ Validar opciones inválidas en el menú  
- 🚪 Salir del programa  

---

🧩 Estructura del programa:

El sistema está organizado en funciones para mejorar la legibilidad y organización del código:

- `agregar_producto()` → Permite ingresar un nuevo producto  
- `mostrar_inventario()` → Muestra todos los productos  
- `calcular_estadisticas()` → Calcula datos del inventario  

Además, utiliza:

- 📚 Listas → Para almacenar los productos  
- 📌 Diccionarios → Para representar cada producto  
- 🔁 Ciclo `while` → Para mantener el menú activo  
- 🔀 Condicionales `if/elif/else` → Para controlar las opciones
